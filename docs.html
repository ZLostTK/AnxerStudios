Guía de Instalación

Sigue estas instrucciones para configurar Anxer Studios en cualquier plataforma y optimizar tu experiencia de streaming.

Introducción Windows (Setup) Ubuntu / Debian Fedora (.rpm) Arch Linux Docker & CasaOS
Nota de Desarrollo La aplicación está en constante evolución. Si encuentras algún comportamiento inesperado, revisa los Releases en GitHub para obtener la última versión estable. En la página principal encontrarás notas de la versión 2.3.0 (abril 2026), incluidas mejoras de reproducción y modo premium.
Sobre la Publicidad AnxerStudios no contiene anuncios invasivos. Solo incluimos banners laterales en nuestra web oficial para el mantenimiento de los servidores. Si ves banners intrusivos durante la reproducción, estos provienen de los proveedores de contenido externos, no de nuestra aplicación.

Windows MSI / EXE

Ideal para Windows 10/11 x64. Soporta video nativo (HLS) mediante WebView2.

1. Descarga AnxerStudios_X.X.X_x64-setup.exe desde GitHub.

2. Ejecuta el archivo y sigue el asistente de instalación.

3. Busca "AnxerStudios" en el menú de inicio para comenzar.

Desinstalación: Puedes hacerlo desde Configuración → Aplicaciones → Aplicaciones instaladas, buscando "AnxerStudios".

Ubuntu / Debian Nativo

Utiliza el motor WebKitGTK nativo de tu sistema para máxima compatibilidad.

1. Asegúrate de tener los codecs necesarios:

sudo apt install ubuntu-restricted-extras gstreamer1.0-plugins-bad gstreamer1.0-libav

2. Instala el paquete descargado:

sudo dpkg -i AnxerStudios_X.X.X_amd64.deb sudo apt-get install -f # Resuelve dependencias si es necesario

Fedora RPM

Usa DNF para instalar el paquete RPM localmente:

sudo dnf localinstall AnxerStudios-X.X.X.x86_64.rpm

Arch Linux Power User

Debido a restricciones de librerías en AppImage, recomendamos el uso de debtap para una instalación nativa.

Opción 1: Conversión Nativa (.deb a PKG)

# Instalar debtap desde AUR git clone https://aur.archlinux.org/debtap.git cd debtap && makepkg -sic # Convertir el paquete de Ubuntu debtap -q AnxerStudios_X.X.X_amd64.deb sudo pacman -Udd anxer-studios-X.X.X-1-x86_64.pkg.tar.zst
Tip Pro Puedes lanzar AnxerStudios desde la terminal como servidor con app --server & y apagarlo con app --stop.

Opción 2: Modo Servidor (AppImage)

Usa el motor de Anxer y visualiza el contenido en tu navegador (Firefox/Brave) para soporte completo de codecs.

./AnxerStudios.AppImage --server # Accede a http://localhost:1234

Docker & CasaOS BETA

Despliega AnxerStudios en tu servidor personal, Raspberry Pi o NAS.

# Levantar servidor con Docker Compose docker compose up -d

Disponible para arquitecturas AMD64 y ARM. Ideal para ser usado permanentemente bajo http://[HOST_IP]:1234.
