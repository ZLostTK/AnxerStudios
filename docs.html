Guía de Instalación

Instala Anxer Studios en tu sistema preferido y comienza a disfrutar del mejor catálogo de streaming.

Windows (Setup/MSI) Linux (AppImage) Linux (DEB/RPM) Modos de Servidor Solución de Problemas

Windows Desktop

Recomendado para usuarios de Windows 10 x64 o superior.

1. Descarga el archivo .exe o .msi desde la página principal.

2. Ejecuta el archivo descargado.

3. Si Windows SmartScreen aparece, selecciona "Más información" y después "Ejecutar".

4. Abre la app desde el escritorio.

Linux AppImage

El formato universal que funciona en casi cualquier distribución sin instalar nada.

1. Da permisos de ejecución:

chmod +x AnxerStudios_X.X.X_x64.AppImage

2. Ejecuta directamente:

./AnxerStudios_X.X.X_x64.AppImage

Nota: Para una mejor experiencia, puedes usar herramientas como AppImageLauncher para integrar la app en tu menú de aplicaciones.

Ubuntu/Debian & Fedora

Instala la aplicación de forma nativa en tu sistema.

Debian / Ubuntu / Kali
sudo dpkg -i AnxerStudios_X.X.X_amd64.deb
Fedora / RedHat / RHEL
sudo rpm -i AnxerStudios-X.X.X.x86_64.rpm

Modos de Lanzamiento Advanced

AnxerStudios incluye funciones avanzadas accesibles mediante terminal (CLI).

Modo Servidor Invisible

anxer-studios --server

Inicia el motor de streaming en segundo plano. Ideal para servidores o navegadores en localhost:3000.

Apagado Rápido

anxer-studios --stop

Cierra la aplicación y todas sus instancias de servidor activas de forma segura.

Solución de Problemas

¿Pantalla en blanco al abrir (Linux)?

Asegúrate de tener instaladas las librerías de WebKitGTK. En Ubuntu: sudo apt install libwebkit2gtk-4.0-37.

¿Error al reproducir audio/video?

Desde la v2.0.2 empaquetamos todos los codecs nativos. Asegúrate de estar usando la última versión (vX.X.X) para tener la mejor compatibilidad.
